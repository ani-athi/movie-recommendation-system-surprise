# 🎬 Movie Recommendation System using Surprise

## 📌 Overview

This project builds a movie recommendation system using collaborative filtering techniques. The model predicts user preferences and recommends movies based on past rating behavior.

## 🧠 Approach

* Collaborative Filtering
* Recommendation Systems

## ⚙️ Workflow

1. Data loading (user-movie-rating dataset)
2. Data preparation using Surprise librar
3. Train-test split using `surprise.model_selection`
4. Model training (collaborative filtering algorithm)
5. Model evaluation using RMSE
6. Generating movie recommendations

## 🧠 Model Used

* Collaborative Filtering Model (via Surprise library)

## 📊 Evaluation Metric

* RMSE (Root Mean Squared Error)

## 🎯 Objective

To recommend movies to users by learning patterns from historical rating data.

## 📈 Result

The model successfully predicts user ratings and recommends relevant movies based on learned user-item interactions.

## 🚀 How to Run

* Install Surprise library (`pip install scikit-surprise`)
* Open the notebook in Google Colab
* Run all cells sequentially

## 📁 Files

* `movie-recommendation-system.ipynb`

## 📚 Key Learnings

* Fundamentals of recommendation systems
* Collaborative filtering techniques
* Using Surprise library for recommender systems
* Evaluating models using RMSE
